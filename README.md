# jsDB

NoSQL nodejs in-memory data structure store.

# Installation :
    $ npm install jsdb-nosql -save
    
# Sample Client  :
  - client.js and theatre.js
  
#  jsDB Server :
```sh
$ const jsDB = require('jsdb-nosql')
$ //jsDB.port = xxxxx; // same should be for client
$ jsDB.server();
```
